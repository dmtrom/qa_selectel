в работе используются фреймворки Playwright и Pytest 

test_homepage отвечает за проверку корректной загрузки страницы

test_registration отвечает за проверку кнопки "Создать аккаунт" и перехода на страницу регистрации.

test_http отвечает за проверку статуса сайта


как запустить локально?
1. Клонируйте репозиторий
2. Установите зависимости pip install -r requirements.txt
3. Установите браузеры playwright install
4. Запустите тесты pytest
